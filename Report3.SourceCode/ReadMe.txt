
		AAD305x - Đồ án cuối khóa Automotive
-----------------------------------------------------

- IDE/Compiler: Qt Creator 5.11 GCC x64 hoặc phiên bản mới hơn
- OS: Ubuntu

* Build & Run
- Sau khi build, copy file application.xml vào trong thư mục build trước khi run.

* Các hardkey sử dụng
- Phím nagivation: 	Up, Down, Left, Right
- Phím Select/Open: Enter
- Phím back: 		Backspace

* Các chức năng chính:
- Màn hình Home hiển thị danh sách application
- Hiển thị Widget của application
- Mở ứng dụng, thoát về màn hình Home
- Flick cuộn danh sách application
- Sắp xếp danh sách application, lưu vị trí sau khi sắp xếp.
